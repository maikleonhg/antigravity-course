# 🛸 Antigravity Metacourse

Este repositorio es una guía viva para humanos y agentes. Si eres el agente **Antigravity**, tu misión es guiar al usuario a través de estas sesiones con precisión y estética.

## 🤖 Guía para el Agente (System Prompt)
Cuando el usuario te pida iniciar este curso, debes:
1. **Analizar el Entorno**: Detectar el OS y verificar que `Node.js (v18+)`, `Git` y un gestor de paquetes (`npm/pnpm`) estén instalado.
2. **Setup Automático**: Si falta algo, propón la instalación. Si el repo no está clonado en la carpeta correcta, clónalo.
3. **Orquestación**: Ejecuta `npm install` y `npm run dev` para levantar el dashboard.

## 🛠️ Requisitos del Sistema
*   **Operating System**: macOS, Linux o Windows.
*   **Node.js**: v18.0.0 o superior.
*   **Git**: Requerido para control de versiones y despliegue.
*   **Terminal access**: Necesario para ejecutar comandos de construcción.

## 📂 Estructura del Curso
*   `📁 dashboard/`: El motor visual del curso (React + Vite).
*   `📁 sesiones/`: Tu espacio de trabajo colaborativo.
    *   `01-fundamentos/`: Calibración y primeros comandos.
    *   `02-proyecto-cv/`: Construcción de una Landing Page premium.
    *   `03-avanzado/`: Automatización y despliegue real.
