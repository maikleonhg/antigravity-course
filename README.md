# 🛸 Antigravity Metacourse

Bienvenido al futuro del desarrollo colaborativo con IA. Este no es solo un curso, es una experiencia inmersiva donde aprenderás a dominar a **Antigravity**, tu agente de IA, para construir aplicaciones increíbles en tiempo record.

## 🚀 ¿Cómo empezar?

1. **Clona este repositorio** (si ya estás aquí, ¡vas por buen camino!).
2. **Instala las dependencias**:
   ```bash
   npm install
   ```
3. **Inicia el Dashboard Interactivo**:
   ```bash
   npm run dev
   ```

## 📂 Estructura del Laboratorio

*   `📁 dashboard/`: La aplicación web que guía tu progreso.
*   `📁 sesiones/`: Carpetas de trabajo donde realizarás tus proyectos junto a Antigravity.
    *   `01-fundamentos/`: Domina el arte de hablar con la IA.
    *   `02-proyecto-cv/`: Tu primera landing page premium.
    *   `03-avanzado/`: Automatización y flujos complejos.

## 🛠️ Herramientas Necesarias

*   **Node.js** (v18 o superior)
*   **Git**
*   **Antigravity Agent** (instalado y configurado)
